News About Src Project Ami

Introduction

Ami is a work in progress amiga emulator. It is being developed soley to pass the time. The aim at present is to get an original A500 working in a zero configuration fashion.

As you can probably tell, it is a long way off being a proper emulation. I have only emulated the hardware as far as I needed to get workbench running. I considered that the first step. It is not written for speed (nor accuracy at present), it won't run any software other than the workbench disk.
